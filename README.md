# SimpleInterestCalculator
Simple Interest Calculator
Description
This project is a simple interest calculator that allows users to calculate the interest earned on a principal amount over a specified time period at a given rate of interest.

Formula
The formula for calculating simple interest is: [ \text{Simple Interest} = \frac{P \times R \times T}{100} ] Where:

(P) = Principal amount
(R) = Rate of interest (in percentage)
(T) = Time (in years)
How to Use
Enter the principal amount.
Enter the rate of interest.
Enter the time period in years.
Click on the "Calculate" button to get the simple interest.
